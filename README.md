# 셀 퀘스트 : 오피스 몬스터 도감

엑셀 스프레드시트로 위장한 텍스트 수집형 RPG.

## 플레이
GitHub Pages 주소로 접속하면 바로 플레이할 수 있습니다.
- 셀 클릭 또는 숫자키(1~9)로 진행
- **ESC** : 가짜 연차 계획 시트로 즉시 위장 전환 (셀을 클릭해 직접 편집도 가능)
- 💾 **자동 저장**: 진행 상황이 브라우저 로컬 캐시(localStorage)에 자동 저장됩니다.
  다음 접속 시 타이틀 화면의 **이어하기**로 바로 복원할 수 있어요.
- **백업 코드**: 다른 기기/브라우저로 옮기거나 캐시 삭제에 대비할 때 사용하세요.
  메뉴에서 코드 발급 → 메모장 등에 보관 → 붙여넣어 복원 (구버전 코드도 호환)

## 게임 내용
- **70종** 오피스·이세계 몬스터 도감 (포획 / 진화 / 합성 / 돌연변이)
- 개체 희귀도: 일반 → **황금**(6%) → **프리즘**(1.5%) → **레전드**(0.3%)
  - 등급이 높을수록 능력치 상승 & 포획 난이도 상승
  - 레전드 개체는 전용 패시브 특성 보유 (흡혈 / 강철 / 집중 / 재생)
- **1부 (오피스)**: 5개 층 + 보스 5인 + 엔딩 후 야근차원 & 전설 포획
- **2부 (이세계)**: 진엔딩 후 과로사… 그리고 전생! 5개 지역 + 보스 5인 + 마왕 토벌
  - 이세계 몬스터는 **2단 진화**, 진화 시 드물게 **돌연변이** 특별종 탄생
  - 난이도 대폭 상승 — 도핑·합성 없이는 클리어가 어렵습니다
- **도핑**: 수상한 앰플로 영구 강화 도전 (실패 시 영구 약화 — 도박!)
- **합성 실험실**: 동종 합성(등급 상승 도전) / 이종 합성(레시피로 신종 창조)
- 상점, 보관함(페이지·풀어주기), 타입 상성 (카페인 > 서류 > 전자 > 카페인 / 야근)
